# AMBILUM
Ambilum is a centralized PC-based ambient automation system that adjusts RGB lighting and monitor brightness based on environmental light, time, and user behavior.
## Features
1. Autobrightness for pc components, WLED devices and Monitor.
2. Automatic Scene Dectection(e.g. Productivity, Gaming, Idle etc)
3. Custom Preset and Scene Saving.
4. Scheduling.
5. TBA...
## Hardware Requirements
1. ESP-8266(or any other Arduino C++ compatible board)
2. BH1750 Sensor
3. WLED Ready Device(optional)
## Software Requirements
1. Twinkle Tray(for controlling monitor brightness)(optional)
2. OpenRGB(For PC component device)(optional)
